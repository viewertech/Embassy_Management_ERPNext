## Presentation Notes

The deck intentionally uses generic diplomatic and consular visual language instead of any official national flag, coat of arms, mission seal, or government emblem. Add approved official branding assets only after they are supplied and authorised by the client mission.

Before a production sign-off presentation:

1. Load demo data only in a demo or pilot environment.
2. Remove `EMSDEMO` demo records after the presentation.
3. Rotate any exposed passwords before production.
4. Confirm backup and restore were tested.
5. Confirm the public portal hostname, privacy notice, declaration text, and payment process are approved by the mission.

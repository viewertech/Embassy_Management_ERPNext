# Configuration Guide

[Powered by Viewertech](https://viewertech.net)

## Embassy Settings

Configure mission identity, countries, location, public emails, default currency, languages, timezone, appointment windows, holiday list, branding, leadership, privacy notice, declaration text, and submission confirmation.

## Consular Services

Create a Consular Service for each service offered by the mission. Use sample services only as examples. Configure category, public visibility, appointment requirement, online application requirement, document upload requirement, payment requirement, review requirement, biometrics, interview, fee item, default fee, workflow, print format, and email template.

## Forms

Build Application Form Templates from sections and fields. Use Application Form Rules for conditional notices, document requirements, appointment requirements, eligibility restrictions, and fee logic.

## Fees

Use ERPNext Items as fee products. Use Consular Fee Rule only for embassy-specific matching by service, nationality, residence country, processing type, entry type, and validity dates.

## Appointments

Configure Appointment Location, Appointment Service Type, Appointment Officer, and Appointment Slot. Slots can be used for internal booking or mapped to external booking references in Embassy Appointment.

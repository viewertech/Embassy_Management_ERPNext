# Appointment Officer Guide

[Powered by Viewertech](https://viewertech.net)

This guide is for staff who manage consular appointment capacity and attendance.

## Configure Appointment Locations

Open:

```text
Appointment Location
```

Create each public counter, room, or service location. Set the time zone and active status.

## Configure Appointment Service Types

Open:

```text
Appointment Service Type
```

Link each appointment type to a Consular Service where appropriate. Configure:

- Duration minutes
- Default capacity
- Requires application
- Enabled status

## Configure Appointment Officers

Open:

```text
Appointment Officer
```

Link each officer to a User record. Set the default location and active status.

## Create Appointment Slots

Open:

```text
Appointment Slot
```

Create slots with:

- Service
- Location
- Officer
- Slot date
- From time
- To time
- Capacity
- Holiday exclusion

Slots can support multiple bookings if capacity is greater than one.

## Manage Bookings

Open:

```text
Embassy Appointment
Appointment Calendar
```

Use bookings to:

- Confirm appointment date and time
- View booking code
- Link to the Consular Application
- Reschedule
- Cancel
- Mark completed
- Mark no-show
- Register walk-ins
- Assign queue numbers

## External Booking References

If the mission uses an external booking system, store the external reference on the Embassy Appointment. The app does not depend on any single provider.

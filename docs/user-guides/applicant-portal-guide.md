# Applicant Portal Guide

[Powered by Viewertech](https://viewertech.net)

This guide is for applicants using the public Embassy Management portal.

## Access The Portal

Open the public mission portal:

```text
/embassy
```

Use the portal to check requirements, start applications, upload documents, book appointments, view payments, read messages, and track case status.

## Check Requirements Before Applying

Open:

```text
/embassy/requirements
```

Select:

- Nationality
- Residence country
- Service or visa type
- Entry type or processing type where applicable

The page shows eligibility, fee, processing time, document requirements, appointment requirement, notices, and the Apply button. Requirements are rule-based and configurable by the mission.

## Register And Sign In

Create a website user account using the email address you will use for the application. Complete email verification if enabled by the mission.

Use the same account for:

- Draft applications
- Submitted applications
- Appointment bookings
- Applicant messages
- Payment status
- Downloads

## Start A New Application

Open:

```text
/embassy/apply
```

Choose the consular service and complete each section:

1. Booking
2. Personal Details
3. Contact Details
4. Passport Details
5. Travel Details
6. Documents
7. Review & Submit

Use Save Draft if you need to return later.

## Upload Documents

Upload only the documents requested by the portal. Each upload links to the secure Frappe File record for your application.

Accepted file types and maximum size depend on the service. If an officer marks a document as unclear, expired, rejected, or missing, upload the corrected document from your dashboard.

## Book An Appointment

Some services require an appointment. If required, select an available appointment slot and keep the booking code.

The mission may reschedule, cancel, or mark no-show appointments according to published policy.

## Review And Submit

Before submission:

- Review all sections.
- Confirm required documents are uploaded.
- Read the privacy notice and declaration.
- Accept the terms and declaration.

After final submission, the application is locked for applicant editing unless the mission requests additional documents.

## Track Progress

Open:

```text
/embassy/dashboard
```

Track:

- Draft applications
- Submitted applications
- Application status
- Payment status
- Appointments
- Messages
- Receipts and downloads

Contact the mission through the published enquiry channel if you cannot access your account.

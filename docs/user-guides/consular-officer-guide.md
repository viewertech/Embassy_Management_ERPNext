# Consular Officer Guide

[Powered by Viewertech](https://viewertech.net)

This guide is for consular, visa, passport, and case-review officers.

## Open The Officer Workbench

In ERPNext Desk, open:

```text
Officer Workbench
```

Use filters for:

- Service
- Status
- Priority
- Appointment date
- Assigned officer

Open each Consular Application from the workbench or from the Embassy Management workspace.

## Review Application Details

Confirm:

- Applicant profile
- Service
- Submission channel
- Priority
- Appointment status
- Payment status
- Service-specific details
- Applicant notes

Use standard ERPNext Communication for applicant-facing messages. Use internal notes for staff-only case comments.

## Review Documents

Open linked uploaded documents from:

```text
Application Uploaded Document
```

Set the review status:

- Accepted
- Rejected
- Unclear
- Expired
- Missing
- Re-upload Requested

Add review notes when requesting corrections. The audit trail records reviewer and date.

## Request Additional Documents

If the application is incomplete:

1. Set application status to Awaiting Additional Documents.
2. Mark the relevant document as Re-upload Requested or Missing.
3. Send an applicant-facing Communication using the configured template.
4. Add internal notes if escalation or supervisor visibility is needed.

## Make A Recommendation

After document and case review:

- Approve where delegated.
- Reject with a clear reason.
- Escalate to supervisor where required.
- Reassign if the case belongs to another queue.

Decision letters, certificates, visas, attestations, or outputs should use configured print formats and official letterheads supplied by the mission.

## Close The Case

Move completed cases to:

```text
Completed
Collected / Dispatched
```

Use the final status that matches the mission process. Keep applicant-facing comments professional and concise.
